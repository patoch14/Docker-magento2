# Docker-magento2

### Initialisation du docker
1 . Copier le dump dans le repertoire sql

2 . Aller dans le repertoire html & cloner le dépot Magento2 dedans (**Le dépot doit s'appeler magento2**)

3 . Lancer le docker (>docker build)

4 . Le dev devrait être accessible sur le vhost config dans le ficher site.conf
